# Kezelési útmutató - Cube 3D Grafika Alkalmazás

## Mód választás (número billentyűk)
- **1** - Egy modell (nézet: automatikus forgás)
- **2** - 4×4-es rács (24 kocka)
- **3** - Két modell, második forgás automatikus
- **4** - Két modell, interaktív (mindkettő módosítható)

## Kamera vezérlés
- **W** - Előre
- **A** - Balra
- **S** - Hátra
- **D** - Jobbra
- **Egér** - Nézés forgatása (drag)

## Objektum 1 vezérlés
- **Q / E** - Forgatás (Y tengely)
- **← Nyíl** - Balra
- **→ Nyíl** - Jobbra
- **↑ Nyíl** - Fel
- **↓ Nyíl** - Le

## Objektum 2 vezérlés (amikor elérhető)
- **U / I** - Forgatás (Y tengely)
- **O / P** - Balra / Jobbra
- **J / K** - Fel / Le

## Anyag és fény beállítások
- **F2** - Ambiens fény (szürke)
- **F3** - Diffús fény (fehér)
- **F4** - Spekuláris fény (fényes)
- **F5** - Réz anyag (fém hatás)
- **F6** - Arany anyag (arany hatás)
- **+/-** - Diffusza красная komponens módosítása
- **F1** - Ez a súgó megjelenítése

## Vezérlők
- **ESC** - Kilépés
- **F1** - Súgó megjelenítése

## Mód részletek

### Mód 1 - Egy modell
- A modell automatikus forgáson halad
- Q/E billentyűkkel manuálisan forgatható
- Pozíciókezelés az nyílbillentyűkkel

### Mód 2 - 4×4 rács
- 16 kocka 4×4-es elrendezésben
- Első modell nagyítható/kicsinyítheto

### Mód 3 - Két modell, forgás
- Első modell (kocka) statikus
- Második modell (macska) automatikus forgáson halad

### Mód 4 - Interaktív
- Mindkét modell teljes interaktív vezérlő
